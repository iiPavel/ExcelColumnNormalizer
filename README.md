# Excel 列标准化工具 (Excel Column Standardization Tool)

这是一个桌面实用工具，旨在通过根据指定模板重排 Excel 文件的列顺序并自动填充缺失列，来实现文件的标准化处理。

## 主要功能
- **批量处理**：支持一次性转换多个文件或整个文件夹。
- **模板管理**：支持创建、保存和加载列模板。
- **智能填充**：自动用空值填充缺失的列。
- **灵活输入**：支持 `.xlsx`、`.xls` 和 `.csv` 格式。
- **标准化输出**：输出文件统一为 `.xlsx` 格式，且列顺序严格与模板一致。

## 安装指南

1. 确保已安装 Python 3.9+。
2. 创建虚拟环境（可选但推荐）：
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # macOS/Linux
   # .venv\Scripts\activate   # Windows
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 启动应用程序：
   ```bash
   python main.py
   ```
2. **选择文件**：使用“添加文件”或“添加文件夹”按钮选择源 Excel 文件。
3. **配置模板**：
   - 手动输入列名称（每行一个）。
   - 或点击“从文件提取”直接读取现有 Excel 文件的表头。
   - 使用“保存模板”将常用模板保存为文件以备后用。
4. **开始处理**：
   - 选择输出目录（可选，默认为 `源文件目录/converted`）。
   - 点击“开始处理 (START PROCESSING)”。
   - 查看日志以了解进度和任何错误信息。

## 故障排除
- 如果输出文件中缺少列，请确保它们已列在模板中。
- 如果应用程序无法启动，请检查 `PySide6` 是否已正确安装。
